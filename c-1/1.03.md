Define a procedure that takes three numbers as arguments and returns the sum of the squares of the two larger numbers.

```
(define (square x) (* x x))

(define (larger x y z count)
        (if (= count 0)
            (+ (square z) (square y))
            (cond ((> x y) (larger y z x (- count 1)))
                  (else (larger x z y (- count 1)))
                ))
    )

(define (f x y z)
    (larger x y z 2)
)

(f 1 3 2)
```
